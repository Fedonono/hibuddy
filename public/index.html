Hi Buddy!

Damn simple video chats for the web!

HiBuddy is a damn simple web application to make video chats with your friends. Create a room, give the url to someone else and say hi!

No signup

Create a room right away, no sign up needed, no pricing or any premium plan bullshit.

Privacy friendly

We do not keep any data about you or your friends.

Free as in Freedom

HiBuddy is a free software, licensed under the GNU Affero General Public License v3.

Install your own

HiBuddy is easy to install, checkout the source from github and deploy in a minute.

Made with love

HiBuddy is made with HTML5 technologies, Node.js and WebRTC.

Code by @tOkeshu, design and css with the help of twitter bootstrap.
